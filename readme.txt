即时认证之二：
	在执行任何一种Action的操作之前，对所有操作的方法，进行即时认证。
	- 编写AuthInterceptor，在这个拦截器中，获得类名、方法名这些信息
		* 不能根据类前面是否定义了@Res注解以及方法前面是否定义了@Oper注解来判断
		  因为定义注解并不是必须的！我们也可以不通过注解来定义某个类及方法作为资源/操作
	- 根据类名、方法名，查询获得其对应的资源唯一标识、操作唯一标识
		- 如果找不到对应的资源或操作，则不做控制
		- 如果找到了对应的资源或操作，而且不允许做这种操作，则拒绝；否则允许！
		spring  hibernate struts2 jbpm jquery oa
